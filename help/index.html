  2. Choisir le projet (i.e le site) à modifier.

UTILISATION GENERALE : Passer d'une page à l'autre

  1. Une fois les modifications effectuées sur une page, sauvegarder avec le bouton 'Save' de la barre d'outils.

  2. Pour changer de page il faut se rendre sous l'URL octoboot.ovh. Dans un rectangle grisé il y a les URLs internes au site qui se présentent sous la forme /Iberia/nomdelapage/index.html. Il suffit de cliquer sur le rectangle et de choisir la nouvelle page à modifier.

  3. A noter : la home page se nomme directement /Iberia/index.html

UTILISATION GENERALE : Insérer un lien sur un bouton

  1. Passer en mode édition 'edit' sur la barre d'outils. Cliquer sur le bouton sur lequel ajouter un lien. Dans la barre de sélection, cliquer sur le logo représentant 2 maillons 'link'

  2. Il existe 2 sortes de liens:
    - Les liens internes au site. Paramétrée par défaut, la fenêtre de défilement des différentes pages du site s'ouvre. Sélectionner la page ou le fichier de destination. Celle-ci peut soit s'ouvrir dans le même onglet (fenêtre) que la précédente --> option 'self' soit dans un nouveau --> option blank. Pour les sites Renaissance Spirits il est recommandé de choisir l'option blank'.
    - Les liens externes au site. Pour y accéder il suffit de cliquer en dehors du menu déroulant qui s'ouvre par défaut à l'ajout d'un lien. Dans la fenêtre http il suffit de coller le lien. Valider. Par convention et pour le confort de l'internaute, les liens externes s'ouvrent dans un nouvel onglet --> option 'blank'.
     

  3. Quitter le mode édition 'edit'

  4. SAUVEGARDER

UTILISATION GENERALE : Modifier du texte

  1. Dans la barre d'outils à gauche, cliquer sur le mode édition 'Edit'.

  2. SOIT il est possible de modifier directement le texte en double-cliquant sur celui-ci

  3. SOIT il est également possible de modifier en survolant le texte avec la souris et en appuyant sur la touche espace du clavier.

  4. SAUVEGARDER dans la barre d'outils en cliquant sur le bouton 'Save'.

UTILISATION GENERALE: Modifier une image

  1. Préparer le(s) fichier(s) image à ajouter (ou modifier). Attention au nom des fichiers, il est préférable d'éviter les espaces, les apostrophes, les caractères spéciaux,... 

  2. Uploader à l'aide du bouton 'Upload' de la barre d'outils, le(s) fichiers dans la bibliothèque du backoffice.

  3. Dans la barre d'outils, passer en mode édition 'Edit'. Au clic sur l'image, une barre de sélection apparaît, cliquer sur l'icone image 'update image'. Rechercher l'image dans la bibliothèque en utilisant le champs de recherche.
    Remarque :  le champs de texte alternatif sert au référencement Google.

  4. SAUVEGARDER

UTILISATION GENERALE: Supprimer un fichier image uploadé dans la bibliothèque

  1.  Cliquer sur la barre de navigation interne (/iberia/.... ou /RenaissanceSpirits/...) comme pour changer de page.

  2. Dans le menu déroulant qui s'ouvre, les images uploadées apparaissent également, /Iberia/uploads/NOMDELIMAGE.png par exemple. Sélectionner le bon fichier. Il est possible d'utiliser uniquement le nom du fichier pour le retrouver.

  3. Le fichier image s'ouvre

  4. Dans la barre d'outils à gauche, cliquer sur 'Delete'

  5. SAUVEGARDER

AJOUTER UN CONTACT

  1. Les éléments du site les plus importants sont duplicables. Il suffit de passer en mode duplication 'Duplicate' dans la barre d'outils. Une barre de sélection apparaît sur les éléments duplicables, soit un signe plus 'Duplicate' soit une croix 'Delete'. Cliquer sur le plus, le nouvel élément dupliqué se place automatiquement.

  2. Préparer le(s) fichier(s) image.

  3. Uploader le(s) fichier(s) dans la bibliothèque avec le mode 'Upload' de la barre d'outils.

  4. Passer en mode édition 'Edit', cliquer sur l'image à modifier et cliquer dans la barre de sélection sur 'Update image'. Sélectionner le bon fichier en utilisant le champs de recherche. 

  5. Modifier le texte SOIT en double cliquant dessus SOIT en le survolant avec la souris et en appuyant sur la touche espace du clavier.

  6. SAUVEGARDER

AJOUTER UNE NOUVELLE ACTUALITE

  1. Dans la barre d'outils, passer en mode duplication 'Duplicate'. Les différents éléments duplicables du site apparaissent, actualités comprises. En fonction de l'ordre souhaité dans cette rubrique, dupliqué une des 3 actu. 

  2. Quitter le mode 'Duplicate' et passer en mode édition 'Edit' dans la barre d'outils. Modifier les images et le texte. Compte tenu des clicks et de l'animation de cette rubrique, il est conseillé de modifier les titres principaux en survolant avec la souris le texte puis en appuyant sur la barre d'espace. 

  3. SAUVEGARDER

AJOUTER UNE NOUVELLE ANNEE A LA RUBRIQUE ACTUALITE

  1. Dans la barre d'outils, passer en mode duplication 'Duplicate'. Les différents éléments duplicables du site apparaissent, repères des années des actualités compris. Dupliqué la dernière année.
    Remarque: Toute la dernière année est dupliquée, ce qui comprend l'intégralité des actualités, il suffit ensuite d'ajuster le nombre d'actualité au contenu de la nouvelle année.

  2. Quitter le mode 'Duplicate' et passer en mode édition 'Edit' dans la barre d'outils. Modifier les images et le texte, supprimer des actualités si il y en a trop.

AJOUTER UNE NOUVELLE MARQUE

Avant de commencer, il est recommandé de préparer un fichier contenant l'ensemble des éléments nécéssaires à l'ajout d'une marque : images ( vérifier les tailles), les contenus, les liens, la couleur du background

  1. Dans la barre d'outils, passer en mode duplication 'Duplicate'. Les différents éléments duplicables du site apparaissent, les marques comprises. 

  2. Aller dans le Menu>Marques>La catégorie d'alcool de la marque a ajouter. Sélectionner une marque déjà existante et cliquer sur le "+" pour dupliquer.  Les espaces et les animations de la nouvelle marque sont créés partout il ne reste plus  qu'à les modifier

  3. Quitter le mode 'Duplicate' et passer en mode édition 'Edit' dans la barre d'outils. Modifier le texte dans le menu. Compte tenu des clicks et de l'animation de ce menu, il est conseillé de le modifier en survolant avec la souris le texte puis en appuyant sur la barre d'espace. 

  4. Pour rajouter la nouvelle bouteille dans l'animation des marques. Passer en mode Upload pour charger l'image de la bouteille dans la bibliothèque du back office. Attention à bien utiliser une image de bonne qualité et comportant une ombre .

  5. Repasser en mode édition 'Edit'. Au moment de la duplication, une bouteille a été ajouté dans l'animation bouteille. Remplacer l'image de la bouteille à partir de la barre de sélection et du bouton 'Upload image'. 

  6. Il faut maintenant placer la bouteille dans l'animation. Sélectionner la bouteille, dans la barre de sélection, cliquer sur l'icone à gauche représentant un crayon 'Style CSS'. Une barre de mise en forme apparait à gauche à la place de la barre d'outils. Il est possible de modifier le positionnement de l'image selon un axe horizontal et vertical, ainsi que la taille de la bouteille en pixel. Modifier d'abord la taille puis positionner la bouteille en utilisant les curseurs 'Top' et 'Left'

  7. SAUVEGARDER

MODIFIER UNE FICHE PRODUIT OU LE PACK DE PHOTOS DES PRODUITS

  1. Uploader le nouveau fichier PDF ou le dossier zip de photos dans la bibliothèque en cliquant sur le bouton 'Upload' de la barre d'outils.

  2. Passer en mode édition 'Edit'. Cliquer sur le bouton de téléchargement de la fiche produit ou des photos. Cliquer sur le bouton 'link' de la barre de sélection.

  3. Rechercher le nouveau PDF ou le fichier zip dans la bibliothèque en utilisant le champs de recherche. Valider et choisir une ouverture en mode 'blank' (i.e. dans une nouvelle fenêtre de navigation).

  4. SAUVEGARDER

AJOUTER UNE NOUVELLE LANGUE

  1. Avec la barre de navigation interne (/nomdusite/nomdelapage) charger la page que vous souhaitez traduire.

  2. Dans la barre d'outils à gauche, cliquer sur 'Plugins' puis sur 'Language'. L'ensemble du texte de la page se charge dans des blocs indépendants.

  3. Dans le menu de ce plugin à gauche, cliquer sur 'New' pour ajouter un nouveau language, sélectionner la langue dans le menu déroulant puis valider.

  4. Les blocs sont alors dupliqués, à gauche le texte dans le language inital (par défaut) à droite le texte à ajouter. Il suffit de remplir les blocs qui apparaissent alors en 'Translated' puis de sauvegarder

  5. REMARQUE 1: Si le texte n'a pas besoin d'être modifié (ex: Renaissance Spirits ou Maison Villevert), ne rien remplir dans la colonne de droite. 

  6. REMARQUE 2:  Certains blocs de textes peuvent se répéter, il ne s'agit pas d'un bug mais d'une redondance dûe à la mise en page et/ou à la version mobile

  7. REMARQUE 3: A la sauvegarde dans le plugin celui-ci se ferme et recharge la page avec le language par défaut pour continuer à modifier le nouveau language il suffit de re-cliquer sur le 'Plugins' puis 'Language'

AFFICHER LA VERSION D'UNE PAGE DANS UNE AUTRE LANGUE

  1. Une fois un language créé et sauvegardé, à la recharge de la page, apparait à droite de la barre de navigation interne un nouveau menu déroulant 'Select Language'

  2. Sélectionner le language désiré, le texte de la page se recharge

  3. REMARQUE 1 : Dans ce mode, l'édition est limitée pour éviter au maximum les bugs et les conflits. Vous pouvez modifier les liens internes ou externes, le texte ainsi que les images

MODIFIER LE TEXTE D'UNE AUTRE LANGUE

  1. Une fois un language créé et sauvegardé le plugin se ferme et le backoffice recharge la page avec le language par défaut.

  2. Pour modifier de nouveau le texte dans le nouveau language:
    soit : il suffit d'ouvrir de nouveau le plugin 'Language' dans la barre d'outils et d'utiliser la fonction recherche ctrl+F
    soit:  charger la version de la page dans la nouvelle langue, passer en mode édition et double cliquer sur le texte à modifier, le plugin  se rouvre alors à l'endroit voulu.

PUBLIER VOTRE SITE SUR UNE PAGE DE TEST

  1. Avant une mise en ligne effective du site, il convient de tester les modifications sur une page de test.

  2. Dans la barre d'outils, cliquer sur 'Publish', puis sur Github.

  3. Vérifier le site à l'adresse indiquée à la fin de la publication

MISE EN LIGNE DU SITE

  1. Une fois les modifications vérifiées. Vous pouvez procéder à la mise en ligne effective du site

  2. Dans la barre d'outils, cliquer sur 'Publish', puis sur 'Custom FTP'.

  3. Rentrer UNIQUEMENT le nom du FTP, l'identifiant, le mot de passe puis remote path france pour RSF et iberia pour iberia (attention pas de majuscule)

  4. Remarque: la mise à jour effective du site peut prendre plusieurs minutes. Avant de vérifier si la nouvelle version du site est bien en ligne, vider votre cache navigateur
